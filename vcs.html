What is VCS ?

To give you more material for a better insight into this system, I will provide you with an explanation about how the so-called Version Control System works.
Version Control System - VCS will manage and store all the files of the entire project and also the modifying history of each file.
With each modification, a version will be created .
The system will help developers to clone and store different versions of source code from the repository,
each change of source code from local devices will be committed and posted to the server where the repository is located .
